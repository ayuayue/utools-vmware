# utools-wmare
utools插件: 快速开启虚拟机

使用`npm install`下载包，之后再`webpack`进行打包。
utools 的插件代码位于dist，导入即可。

# 使用说明
有三种指令，`vminit`用于配置虚拟机文件夹地址，`vmopen`用于打开虚拟机，`vmrun`用于启动虚拟机。同时，在`vmopen`和`vmrun`处，支持查询。
